================================================================================
  NOVAS FUNCIONALIDADES IMPLEMENTADAS
================================================================================


1. PRÉ-MENU
--------------------------------------------------------------------------------
- Nova tela inicial antes da configuração
- Campo de nome do jogador (salvo no localStorage)
- Botão "Modo Offline" → leva ao draft como era antes
- Botão "Multiplayer" → abre menu de salas
- Painel de Opções com alternador Light/Dark Mode
- Estrutura para Som (placeholder)

2. DARK MODE
--------------------------------------------------------------------------------
- Tema completo via CSS custom properties
- Alternador com ícone dinâmico (🌙 Modo escuro / ☀️ Modo claro)
- Persiste no localStorage
- Logo alternativo para dark mode (logo-darkmode.svg)

3. COUNTDOWN TIMER
--------------------------------------------------------------------------------
- 10 segundos para seleção de países (auto-pick aleatório no timeout)
- 15 segundos para draft (auto-pick aleatório no timeout)
- Timer visível no header centralizado
- Pulsa nos últimos 3 segundos
- Para ao clicar e reinicia para o próximo turno

4. MULTIPLAYER ONLINE (via Supabase)
--------------------------------------------------------------------------------
Backend:
- Conta no Supabase (https://supabase.com)
- Tabelas: rooms, room_players
- Realtime Presence (jogadores online)
- Realtime Broadcast (picks, pool, mata-mata)
- Postgres CDC (mudanças na sala)

Funcionalidades:
- Criar sala com código de 6 dígitos
- Entrar na sala pelo código
- Lobby com lista de jogadores online
- Moderador configura as regras do draft
- Draft sincronizado em tempo real
- Turnos respeitados (só quem é a vez pode agir)
- Pool idêntico para todos os jogadores
- Timer só roda para quem é a vez
- Mata-mata restrito (só moderador pode sortear/editar)
- Reconexão após F5 (estado preservado no Supabase)
- Kick ao sair (moderador sai → todos voltam ao menu)

Arquivos criados:
- multiplayer.js     → Lógica de salas, lobby, sincronia
- supabase-config.js → Configuração do cliente Supabase
